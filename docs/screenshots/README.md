## 🎨 Aesthetic Guidelines

To maintain the "top 1% quality" look of DayVault documentation, please follow these screenshot rules:

1. **Resolution**: Use high-resolution device exports (e.g., 1080x1920 or higher).
2. **Device Frames**: Ideally, use framed screenshots (iOS/Android device shells).
3. **Consistency**: Use the same device/ratio for all screenshots in a set.
4. **Dark Mode**: Since DayVault is dark-first, ensure screenshots highlight the glassmorphism.
5. **Format**: Use `.png` or `.webp` for maximum quality.
